# Insights

## Key Observations

- Multiple employees can share the highest salary
- Titles must be joined from a separate table
- Subqueries are useful for scalar comparisons

## SQL Learnings

- Avoid unnecessary joins in subqueries
- MAX() is commonly used for filtering top values
- JOIN + subquery is a core SQL pattern

## Business Insight

Understanding top earners and their roles helps guide compensation strategy and organizational planning.
